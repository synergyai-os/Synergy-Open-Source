# Cursor Keyboard Shortcuts

> **Quick Reference**: Essential keyboard shortcuts for faster AI development in Cursor

---

## Code Selection

- `Cmd+Shift+Left/Right` - Select to line start/end
- `Cmd+L` - Add selection to chat

---

## Code Review

- `Cmd+Y` - Accept current change
- `Cmd+Enter` - Accept all changes
- `Option+J` - Cycle forward through changes
- `Cmd+K` - Cycle backward through changes
- `Cmd+N` - Reject current change

---

## Quick Edit

- `Cmd+Shift+K` - Quick inline edit (select code first)

**Usage**: Select code → `Cmd+Shift+K` → Type change → `Cmd+Enter` to accept

---

## Chat Management

- `Cmd+T` - New chat window
- `Enter` - Send message (queued)
- `Option+Enter` - Send instantly (skip queue)
- `Ctrl+C` - Cancel current operation

---

## Mode Switching

- `Cmd+.` - Cycle through modes (agent, chat, terminal)

---

## Tips

**Faster Code Review**:

1. `Option+J` to cycle through changes
2. `Cmd+Y` to accept, `Cmd+N` to reject
3. `Cmd+Enter` to accept all when done

**Quick Edits**:

1. Select code
2. `Cmd+Shift+K` for inline edit
3. Type change
4. `Cmd+Enter` to accept

**Multitasking**:

- `Cmd+T` to open new chat window
- Work on multiple tasks in parallel
- Review code as it comes in
