## ðŸŽ¨ Component Layers

### Atoms

Single interactive elements with design tokens applied.

**Examples**: Button, Input, Badge, Card, Icon

**Location**: `src/lib/components/atoms/`

**Storybook**: `Design System/Atoms/ComponentName`

### Molecules

2-3 atoms composed together with layout logic.

**Examples**: FormField (Label + Input + Error), Dropdown (Button + Menu)

**Location**: `src/lib/components/molecules/` (when created)

**Storybook**: `Design System/Molecules/ComponentName`

### Organisms

Complex sections combining multiple atoms and molecules.

**Examples**: Dialog, Modal, Command Palette

**Location**: `src/lib/components/organisms/`

**Storybook**: `Design System/Organisms/ComponentName`

## ðŸ”„ The Cascade

When you change `--spacing-button-x` in tokens:

1. Updates `px-button-x` utility
2. Updates Button component
3. Updates all pages using Button

**No manual updates needed!**
