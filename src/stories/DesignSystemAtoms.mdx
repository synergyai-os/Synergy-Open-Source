## ðŸ“¦ Available Atoms

### Buttons & Actions

- **Button**: Primary, Secondary, Outline variants with sizes
- **IconButton**: Icon-only button for toolbars
- **SplitButton**: Button with dropdown menu
- **Toggle**: Toggle button for on/off states
- **ToggleGroup**: Group of toggle buttons

### Form Inputs

- **FormInput**: Text input with label and error states
- **FormTextarea**: Multi-line text input
- **Checkbox**: Checkbox input with label
- **Switch**: Toggle switch for settings
- **RadioGroup**: Radio button group
- **PinInput**: Multi-digit PIN input (verification codes)
- **Slider**: Range slider input

### Display Components

- **Badge**: Status badge with variants
- **Chip**: Removable chip/tag component
- **Card**: Container card with variants (default, elevated, outlined)
- **Avatar**: User avatar with fallback
- **Icon**: SVG icon component
- **Text**: Typography component with semantic sizes
- **Heading**: Heading component (h1-h6)
- **Label**: Form label component
- **Separator**: Visual divider

### Feedback & Status

- **Loading**: Spinner loading indicator
- **LoadingOverlay**: Full-page loading overlay
- **Progress**: Progress bar
- **Meter**: Meter/gauge component
- **StatusPill**: Status indicator pill
- **Tooltip**: Hover tooltip

### Layout & Structure

- **AspectRatio**: Maintain aspect ratio container
- **ScrollArea**: Scrollable container with custom scrollbar
