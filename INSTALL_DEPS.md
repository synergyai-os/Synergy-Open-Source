# Install Documentation Dependencies

Run this command to install the required packages for the documentation system:

```bash
npm install --save-dev mdsvex rehype-slug rehype-autolink-headings shiki fuse.js
```

This will install:
- `mdsvex` - MDX support for SvelteKit
- `rehype-slug` - Auto-generate heading IDs
- `rehype-autolink-headings` - Add anchor links to headings
- `shiki` - Syntax highlighting for code blocks
- `fuse.js` - Fuzzy search for Cmd+K functionality

After installation, you can delete this file.
